רוּץ

rûwts — H7323

Transliteration: rûwts

Pronunciation:

Language: Hebrew

Strong’s: H7323

Derivation: a primitive root;

Definition

to run (for whatever reason, especially to rush)

KJV Translation

break down, divide speedily, footman, guard, bring hastily, (make) run (away, through), post.
