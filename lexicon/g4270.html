προγράφω

prográphō — G4270

Transliteration: prográphō

Pronunciation:

Language: Greek

Strong’s: G4270

Derivation: from G4253 (πρό) and G1125 (γράφω);

Definition

to write previously; figuratively, to announce, prescribe

KJV Translation

before ordain, evidently set forth, write (afore, aforetime)
