κατατρέχω

katatréchō — G2701

Transliteration: katatréchō

Pronunciation:

Language: Greek

Strong’s: G2701

Derivation: from G2596 (κατά) and G5143 (τρέχω);

Definition

to run down, i.e. hasten from a tower

KJV Translation

run down
