τρέχω

tréchō — G5143

Transliteration: tréchō

Pronunciation:

Language: Greek

Strong’s: G5143

Derivation: apparently a primary verb (properly, ;

Definition

compare G2359 (θρίξ)); which uses (the base of G1408 (δρόμος)) as alternate in certain tenses; to run or walk hastily (literally or figuratively)

KJV Translation

have course, run
